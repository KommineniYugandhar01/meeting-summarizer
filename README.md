📌Proactive Meeting Summarizer: Overview:
The Proactive Meeting Summarizer is an AI-based project.
It converts meeting audio into text automatically.
Helps reduce manual note-taking during meetings.
Provides a foundation for smart summaries and insights.

🎯 Features:
🎤 Audio generation using text-to-speech
📝 Speech-to-text transcription using AI (Whisper)
💾 Automatic saving of transcription
⚡ Fast processing using lightweight models
🔄 Scalable for advanced features

🛠️ Tech Stack:
Language: Python
Libraries: pyttsx3, Whisper, Torch
Tools: VS Code, GitHub

📂 Project Structure: 
meeting-summarizer/ 
│── generate_audio.py
│── transcribe.py
│── meeting_audio.wav
│── transcription.txt 
│── README.md

📌 Output:
meeting_audio.wav → Generated audio file
transcription.txt → Transcribed text
